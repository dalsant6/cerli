Lucky 8s Proto Client Dashboard

A simple dashboard for Lucky 8s Proto clients to track the status of their prototype orders and store order history.

Build

This dashboard was built using Angular 2.0, Ionic 2, and Cordova frameworks collectively with Bootstrap and Material Design. Auth0 runs user authentication while Firebase handles the database services.

Push Notifications

A simple push notification alerts the user when their order is ready to pickup, or when their parts have shipped.
