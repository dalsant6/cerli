Pokemon GO CP Calculator

A simplified reference tool for the most popular mobile game in history.

Build

Built using the Ionic 2 and Angular 2.0 frameworks, this app is modular and responsive to allow for a wide range of devices and future development.

Community Driven

The calculator runs off the submissions of it's users in order to figure the multiplier for each Pokemon. Hundreds of submitted values are aggregated sans outliers to form a highly accurate value that is fed into a flexible formula derived from Pokemon GO source.
